Abstract

TutorSim is a research project exploring how simulated tutoring conversations can be used to study, evaluate, and improve K-12 math instruction. By pairing language models with grounded representations of student work, we aim to build tools that help teachers and tutors deliver more effective one-on-one support.

This site is a placeholder while the project is in active development. Check back soon for datasets, code, and findings.
